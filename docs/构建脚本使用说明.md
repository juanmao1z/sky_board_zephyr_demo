# 构建脚本使用说明

本文说明如何使用项目内置构建脚本进行跨平台编译。

## 脚本位置

- Linux/macOS: `scripts/build.sh`
- Windows PowerShell: `scripts/build.ps1`

## Linux/macOS 使用方式

```bash
cd /home/zz/zephyrproject/myproject/sky_board_zephyr_demo
source /home/zz/zephyrproject/.venv/bin/activate
./scripts/build.sh
```

## Windows PowerShell 使用方式

```powershell
cd <zephyr-workspace>\myproject\sky_board_zephyr_demo
# 先激活 Zephyr venv
.\scripts\build.ps1
```

## 常用参数

### 1) 强制全量重建

Linux/macOS:

```bash
PRISTINE=always ./scripts/build.sh
```

Windows:

```powershell
.\scripts\build.ps1 -Pristine always
```

### 2) 指定板卡

Linux/macOS:

```bash
BOARD=lckfb_sky_board_stm32f407 ./scripts/build.sh
```

Windows:

```powershell
.\scripts\build.ps1 -Board lckfb_sky_board_stm32f407
```

### 3) 透传 `west build` 参数

Linux/macOS:

```bash
./scripts/build.sh --cmake-only
```

Windows:

```powershell
.\scripts\build.ps1 -- --cmake-only
```

## 烧录命令

```bash
west flash -d build/lckfb_sky_board_stm32f407
```

## 说明

- 两个脚本都会自动向上查找 `.west` 工作区目录。
- 若提示 `west not found`，请先激活 Zephyr Python 虚拟环境。
